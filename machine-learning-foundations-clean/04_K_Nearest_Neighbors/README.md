# K-Nearest Neighbors

Introductory K-Nearest Neighbors exercises covering distance metrics, normalization, manual KNN classification, validation accuracy, and scikit-learn's `KNeighborsClassifier`.

## Files

- `01_representing_points.py`: representing points as lists.
- `02_distance_functions.py`: Euclidean, Manhattan, and Hamming distances from scratch.
- `03_scipy_distances.py`: comparison with SciPy distance functions.
- `04_min_max_normalization.py`: min-max normalization.
- `05_movie_distance_2d.py`: movie similarity in two dimensions.
- `06_movie_distance_3d.py`: movie similarity in three dimensions.
- `07_find_nearest_neighbors.py`: find nearest movie neighbors.
- `08_count_neighbors_classification.py`: classify using neighbor labels.
- `09_classify_new_movie.py`: classify a new normalized movie.
- `10_training_validation_sets.py`: evaluate one validation example.
- `11_choose_k_validation_accuracy.py`: compare validation accuracy for different k values.
- `12_sklearn_knn_classifier.py`: KNN classification with scikit-learn.
- `knn_utils.py`: reusable helper functions.
- `movies.py`: small synthetic movie dataset used by the examples.

## Note

The movie data is synthetic and included only to make the examples runnable outside the original course environment.
