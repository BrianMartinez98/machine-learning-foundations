# Machine Learning Foundations

Curated introductory machine learning exercises with Python, focused on mathematical interpretation, reproducible examples, and clean implementation.

This repository is organized by topic. Each folder contains small scripts that build from basic concepts toward simple machine learning workflows.

## Topics

- Introductory text classification and dataset visualization
- Linear regression
- Logistic regression and classification metrics
- K-Nearest Neighbors from scratch and with scikit-learn
- Planned modules: decision trees, random forests, K-Means clustering, and perceptrons

## Setup

Install the dependencies with:

```bash
pip install -r requirements.txt
```

## Portfolio note

These are introductory exercises, cleaned and organized for review. The goal is not to present advanced production-level machine learning, but to show a clear progression through foundational models, mathematical ideas, and reproducible Python examples.
