# Linear Regression

Introductory linear regression exercises covering scatterplots, manual regression lines, squared loss, gradients, gradient descent, and simple regression with scikit-learn.

## Main concepts

- Scatterplots and visual trend analysis
- Squared loss
- Gradients with respect to slope and intercept
- Gradient descent
- Linear regression with scikit-learn

## Note

Some scripts generate `.png` figures in this folder when run.
