# Logistic Regression and Classification Metrics

Introductory logistic regression exercises covering binary classification, log-odds, the sigmoid function, train/test splitting, classification thresholds, confusion matrices, accuracy, and F1 score.

## Main concepts

- Logistic regression for binary classification
- Linear regression limitations for binary outcomes
- Odds and log-odds
- Sigmoid transformation
- Prediction probabilities
- Classification thresholds
- Confusion matrix, accuracy, and F1 score

## Dataset

`exam_pass_data.csv` is a small synthetic dataset for reproducible examples.
